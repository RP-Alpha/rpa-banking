# rpa-banking

A modern banking system with Glassmorphism UI.

## Features
- **Modern UI**: Clean interface showing balance, transactions, and user info.
- **ATM Integration**: Use Target on ATM models to open banking.
- **Transactions**: Deposit and Withdraw functionalities.

## Installation
1. Ensure `rpa-lib` is started.
2. Add `ensure rpa-banking` to your `server.cfg`.

## Configuration
Database sql file included in `sql/` (if applicable for your framework version).
